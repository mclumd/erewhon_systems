The latest verison of the House program is located in the folder (package) NewHouse.  Alfred information is in NewHouse/Settings.java, while the actual program is started by NewHouse/Driver.java.

Commands:

"person" [] (name) [] (room) [] (x) [] (y)
-Creates a new person with the given name at the given coordinates

"move" []  (Room) [] (X) [] (Y)
-Moves the current person to the specified room at the specified x/y coordinates
