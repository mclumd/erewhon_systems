QUI: An X-based Development Environment

The Quintus User Interface (QUI) is a new development environment based on the X-Windows system. QUI makes use of windows, buttons and menus to increase programmer productivity, and it includes an internal editor, access to GNU Emacs and access to the on-line manual. See qui.
