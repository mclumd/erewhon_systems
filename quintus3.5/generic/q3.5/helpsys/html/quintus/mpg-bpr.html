Built-in Predicates

The following reference pages, alphabetically arranged, describe the Quintus Prolog built-in predicates.

For a functional grouping of these predicates including brief descriptions, see mpg-top.

In many cases the heading of a reference page will contain not only the name and arity of the predicate, but also the name of a major category to which the predicate belongs. These categories are defined in mpg-ref-cat.

Further information about categories of predicates and arguments, mode annotations, and the conventions observed in the reference pages is found in mpg-ref.
