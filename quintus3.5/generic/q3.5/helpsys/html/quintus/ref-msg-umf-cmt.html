Changing message text

Similarly, the text of an existing message may be changed by defining generate_message_hook/3. For example, the following definition will change the text of a default message (see messages(language('QU_messages.pl')) for original text). 

     user:generate_message_hook(no_source(Name, Arity))-->
                   ['~q/~d has no source file' - [Name, Arity], nl].
