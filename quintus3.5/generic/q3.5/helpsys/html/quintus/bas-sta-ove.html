Overview

This chapter describes things that you should know about Quintus Prolog. It assumes only the default interface, the way you can use Prolog on terminals that do not support X-Windows or Emacs.

This section describes how to run and halt Prolog, what you'll see once you've started Prolog, and how to use the on-line help system. bas-lod describes how to load programs into Prolog. bas-run discusses various features of the system related to running programs.
