otherwise/0

Synopsis

otherwise

Always succeeds (same as true/0).

Description

otherwise/0 is useful for laying out conditionals (see ref-sem-con) in a readable way.

Examples

     ( test1 ->
          goal1
     | test2 ->
          goal2
     | otherwise ->
          goal3
     )
