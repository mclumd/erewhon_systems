halt/[0,1]

Synopsis

halt

halt(+ExitCode)

Causes an exit from Prolog.

Arguments


ExitCode integer
an exit status code

Description

causes an exit from Prolog

halt/0 exits with a "success" exit status (0).

halt/1 exits with the exit status given by its ExitCode argument.

Exceptions


instantiation_error
type_error
N is not an integer.

See Also

abort/0, break/0 ref-iex-int
