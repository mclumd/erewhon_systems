is_key/[2,3]
     is_key(?Type, +Key, -Keysym)
     is_key(?Type, +Key)

Identify various subclasses of keys and their keysyms.

Type is one of:


keypad
For keypad keys.
cursor
For cursor control keys.
pf
For pf keys.
function
For function keys.
misc_function
For various other function keys.
modifier
For modifier keys.

Key is the key name.

Keysym is the integer, server-independent, keysym.
