Execution State


abort
abort execution of the program; return to current break level
break
start a new break-level to interpret commands from the user
halt
exit from Prolog
on_exception(E,P,H)
specify a handler H for any exception E arising in the execution of the goal P
raise_exception(E)
raise an exception
