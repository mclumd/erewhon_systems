Built-in Operators
     :-op( 1200, xfx, [ :-, --> ])
     :-op( 1200,  fx, [ :-, ?- ])
     :-op( 1150,  fx, [ dynamic, multifile,
                          meta_predicate, initialization, volatile ])
     :-op( 1100, xfy, [ ; ])
     :-op( 1050, xfy, [ -> ])
     :-op( 1000, xfy, [ ',' ])
     :-op(  900,  fy, [ \+, spy, nospy ])
     :-op(  700, xfx, [ =, is, =.., ==, \==, @<, @>, @=<, @>=,
                                         =:=, =\=, <, >, =<, >= ])
     :-op(  600, xfy, [:])
     :-op(  500, yfx, [ +, -, \/, /\ ])
     :-op(  500,  fx, [ +, - ])
     :-op(  400, yfx, [ /, //, *, <<, >> div  ])
     :-op(  300, xfx, [ mod ])
     :-op(  200, xfy, [ ^ ])

Two additional operators are provided solely for compatibility with other Prologs: 

             :- op(1150, fx, [mode, public])
