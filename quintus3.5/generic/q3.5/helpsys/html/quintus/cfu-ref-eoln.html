QP_eoln()

Synopsis

     #include <quintus/quintus.h>
     
     int QP_eoln(stream)
     QP_stream *stream;

Returns non-zero when the end of record has been reached on the named stream, otherwise zero.

Return Value


QP_ERROR
If an error has occurred
1
If the end of record has been reached on the named stream
0
Otherwise
