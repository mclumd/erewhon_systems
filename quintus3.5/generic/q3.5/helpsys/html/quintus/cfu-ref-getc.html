QP_getc()

Synopsis

     #include <quintus/quintus.h>
     
     int QP_getc(stream)
     QP_stream *stream;

QP_getc() is similar to the library function getc(3V), however it operates on a Prolog stream rather than a stdio. Like getc(3V), QP_getc() is a macro.

Return Value


character read
If function succeeds
QP_ERROR
Otherwise
