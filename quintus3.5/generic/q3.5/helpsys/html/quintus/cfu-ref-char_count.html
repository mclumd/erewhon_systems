QP_char_count()

Synopsis

     #include <quintus/quintus.h>
     
     int QP_char_count(stream)
     QP_stream *stream;

Obtains the current character count for the Prolog stream stream. QP_char_count() is a macro.

Arguments


stream
a pointer to a valid Prolog stream structure
