fileerrors/0

Synopsis

fileerrors

Cancels the effect of nofileerrors/0.

Description

Sets the fileerrors flag to its default state, on, in which an exception is raised by see/1, tell/1, and open/3 if the specified file cannot be opened.

The fileerrors flag is only disabled by an explicit call to nofileerrors/0, or via prolog_flag/[2,3], which can also be used to obtain the current value of the fileerrors flag. See ref-lps-ove, for more information on the fileerrors flag.

See Also

nofileerrors/0, prolog_flag/[2,3]
