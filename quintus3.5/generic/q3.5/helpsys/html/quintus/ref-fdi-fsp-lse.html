Editor Command for Library Search

Under the Emacs interface there is a command <ESC> x library, which, when given FileName as an argument, visits the file specified by library(FileName). This facility is provided so one can visit/edit the copy of the file that will be accessed as a result of a library search.

The notes at the end of ref-fdi-fsp-def apply here as well.
