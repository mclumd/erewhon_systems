QP_peekc()

Synopsis

     #include <quintus/quintus.h>
     
     int QP_peekc(stream)
     QP_stream *stream;

Look a character ahead from a specified Prolog input stream. QP_peekc() is a macro.

Arguments


stream
pointer to a valid stream structure

Return Value

Character code or QP_ERROR.

See Also
