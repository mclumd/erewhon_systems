QP_skipln()

Synopsis

     #include <quintus/quintus.h>
     
     int QP_skipln(stream)
     QP_stream *stream;

Skip the current input record of a Prolog input stream. QP_skipln() is a macro.

Arguments


stream
pointer to a valid Prolog input stream
