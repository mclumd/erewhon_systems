descendant_of/2

Synopsis

descendant_of(+Obj, *Class)

Arguments


Obj
object
Class
atom

Description

Obj is an instance of Class or of a descendant of Class.

Exceptions


instantiation_error
Obj is unbound.
type_error
Object is not a valid object.

See Also

class_ancestor/2, class_of/2, class_superclass/2
