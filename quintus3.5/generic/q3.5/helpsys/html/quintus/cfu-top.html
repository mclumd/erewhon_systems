Following is a complete list of Quintus-supplied C functions. They fall into two categories: built-in and user-redefinable. The user-redefinable functions are used in embedding Prolog sub-programs (see fli-emb). Quintus provides default definitions for these functions, and for most purposes it is unneccessary to even know about them. However, it is possible to redefine any of these to make Quintus Prolog programs behave appropriately when embedded in C code. Changing these default definitions means replacing them with your own code, not adding clauses.

By convention these functions are named to be recognizable as C functions, and as belonging to one or the other of these categories: Regular C builtins are prefixed with QP_ and user-redefinable ones with QU_.
