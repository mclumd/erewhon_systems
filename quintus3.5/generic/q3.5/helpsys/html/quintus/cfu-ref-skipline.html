QP_skipline()

Synopsis

     #include <quintus/quintus.h>
     
     int QP_skipline()

Skip the current input record of the current Prolog input stream. QP_skipline() is a macro.

Description

QP_skipline() is equivalent to QP_skipln(QP_curin).
