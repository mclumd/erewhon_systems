Documentation of Library Packages

  • lib-bas-dlp-acc: Accessing Code Comments

All library packages include code comments, often extensive, which serve as documentation. Accessing these comments and other information available in the library directory is discussed in lib-bas-dlp-acc.

In addition, many packages are more fully documented:

  • The IPC packages are documented in ipc.
  • A large number are documented in lib-lis through lib-mis of this part of the manual.

The rest are abstracted in lib-abs. For these the information in the following section is particularly useful.
