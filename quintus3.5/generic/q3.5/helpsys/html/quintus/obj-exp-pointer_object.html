pointer_object/2

Synopsis

pointer_object(+Addr,-Obj)

pointer_object(-Addr,+Obj)

Arguments


Addr
integer
Obj
object

Description

Addr is the address of object Obj. This can be used to get the address of an object or to get an object given its address.

Exceptions


instantiation_error
both Obj and Addr are unbound.
type_error
Addr is not an integer.
