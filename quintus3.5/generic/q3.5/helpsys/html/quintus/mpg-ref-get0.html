get0/[1,2]

Synopsis

get0(-Char)

get0(+Stream, -Char)

Same as get/[1,2] except that Char includes layout characters.

Arguments


Char char

Stream stream_object

Exceptions

Stream errors (see ref-iou-sfh-est), plus:


existence_error
Trying to read beyond end of Stream

See Also

get/[1,2], ttyget0/1, prompt/[2,3]
